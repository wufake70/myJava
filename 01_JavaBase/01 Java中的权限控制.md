# 01 Java中的权限控制
```
    Java中所有代码都必须写在类中；对于类来说，只有public、包(默认的，)权限，
    public 意味着，该类可以在不同包的不同类 使用。包权限意味着 只能在相同一个包声明下的源文件中使用 该类。

    对于类中属性、方法，则有四种权限public、protected、默认的、private
        public 表示可以在不同包不同类使用
        protected 表示可以在同一个包下使用，以及不同包下继承该类的子类 使用
        默认的 表示可以在同一个包下使用
        private 表示只能在 该类内部使用
```
